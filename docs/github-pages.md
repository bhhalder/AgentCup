# GitHub Pages

This repository can be published with GitHub Pages from the `main` branch root.

## Expected site URL

`https://bhhalder.github.io/AgentCup/`

## How to enable Pages

1. Open repository **Settings**.
2. Go to **Pages**.
3. Under **Build and deployment**, choose:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main`
   - **Folder:** `/ (root)`
4. Save.

GitHub will then publish the site from `index.html` at the repository root.
